What is NAS for?

  • Storing/sharing materials for research, e.g., papers, datasets, codes, trained models, etc.
  • Being linked to workstations for remote development.

DO NOT keep any sensitive information on NAS!


1. Gaining Access To NAS

  • The administrator will provide you with a username and a password to access the NAS.
  • After logging in for the first time, you should change your password via [NAS Web Interface] > [User Logo] > Personal > Account > enter the current password and the new password > OK.

    • [NAS Web Interface] will be introduced in the next section.

2. Connecting To NAS

There are several ways to connect to NAS, each with its own advantages and disadvantages. Try them all and choose the one that best suits your needs.

  1. NAS Web Interface

    2. This method provides a user-friendly interface for accessing NAS via a web browser. Also, for now, it is the only way to manage your NAS account.

    • Open a web browser and go to https://223.195.240.67:1006
    • Enter your username and password.
    • Click Sign In.
    • NAS Web Interface main features include:
      • File Station is for managing files and folders
      • Personal contains your account information
  2. SFTP Clients
    3. Please refer to Data Transfer Between Workstations/NAS for more information.

3. NAS Folder Structure

  • your_username contains your personal data.
  • PAPER-WORK is for storing our lab's documents including salary, invoice, certificates, etc.
  • PUBLIC is for sharing materials from graduated members, this folder is read-only.
